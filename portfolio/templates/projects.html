DineQ

June 2023

  • DineQ, a product of the 2023 Square Developer Hackathon, is a dual-app solution designed to revolutionize the dining experience for both customers and restaurant owners. Developed using Django Rest Framework, Square API, PostgreSQL, and Flutter, it introduces a virtual queue system, in-app ordering, and a digital tab.

  • The customer app, a standout for its virtual queue feature, utilizes geofencing technology and Google Maps API to enable customers to locate nearby restaurants and join a virtual queue from their current location. It eradicates the need for physical queues, cuts down wait times, and streamlines the ordering and payment process through the integration of Square API.

  • The restaurant owner app serves as a robust management tool. It facilitates seamless menu and inventory management, allows for the release of seats for new customers, and ensures secure and efficient payment processing, all made possible by the integration of Square API.

Sssserpent Struggle

April 2023

  • Sssserpent Struggle, developed during the Pygames Devpost Hackathon, is a tribute to the iconic Snake Game that has entertained generations. This project, built using Python and the Pygame library, revives the retro game while introducing new elements to engage today's players. The game is designed to test players' skills and strategy, as they navigate a snake through a playing field, consuming pellets to grow in length. The added challenge is the presence of enemy snakes, also pursuing the same food using a greedy path-finding algorithm.

  • The game stands out for its simplistic yet visually appealing design, smooth and responsive gameplay mechanics, and gradually increasing difficulty as the snake grows in length. The game also features four enemy snakes with advanced path-finding algorithms and blocks that require strategic navigation to avoid or overcome.

Low Latency Cloud Data Pipeline

February 2023

  • The Real-Time Cloud Data Pipeline is a robust, cloud-based solution designed to handle, process, and update millions of real-time transaction data. The primary objective was to create a system that ensures every transaction is processed and stored efficiently, thereby facilitating swift business analytics and reporting.

  • The solution operates on three core components: Data Collection, Data Processing, and Data Analysis. Data Collection uses an API Gateway to accept incoming transaction data, which is then processed and queued. Data Processing involves a Lambda function that inserts the transaction data into an RDS database. Data Analysis uses CloudWatch rules to trigger periodic updates in the Redshift database, providing an up-to-date view of the transaction data for reporting and analytics.

  • The project is distinguished by its scalability and efficiency. It employs an SQS queue-triggered system for efficient data processing and a CloudWatch rules-based system for regular data transfer from RDS to Redshift. This design enables efficient data analysis, cost-effective reporting, and swift dashboard updates, demonstrating the power of AWS services in creating efficient, scalable, and real-time data pipelines for insightful decision-making.

TrailTrack

December 2022

  • TrailTrack, a runner-up project in the Google Developer Student Club (DSC) Hackathon conducted by Northeastern University, is a progressive web application designed to enhance child safety. It uses real-time location tracking technology to provide parents with up-to-date information on their child's whereabouts, offering a powerful tool for safety and awareness.

  • The application serves as a comprehensive safety tool, offering real-time emergency reporting and location tracking capabilities. Its live tracking system ensures parents are constantly updated about their child's location, allowing them to act promptly in case of emergencies. A standout feature of TrailTrack is its push notification system. This system sends alerts to parents if their child moves beyond a predefined boundary, providing an additional layer of security and peace of mind.
